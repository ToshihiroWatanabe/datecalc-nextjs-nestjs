# datecalc-nextjs-nestjs

![image](https://user-images.githubusercontent.com/79039863/147854137-b7813176-d462-4a78-a824-322b540015c9.png)

年月日の加減算アプリ

# 主な使用技術

-   Next.js

-   Apollo Client

-   MUI

-   NestJS

-   Prisma

-   MySQL

-   Docker

# インストールと実行

1.  Docker Desktopを起動して、以下のコマンドで実行

    docker-compose up

2.  <http://localhost:3000/>にアクセス
